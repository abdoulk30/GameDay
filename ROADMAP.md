# ROADMAP.md

## Phase 1: Planning (1–2 days)
- Lock leagues: NBA + NFL
- Pick primary API
- Design final MVP schema (fields and structure)
- Decide tech stack

## Phase 2: Backend Setup (2–3 days)
- Set up backend framework (Node.js / Python)
- Integrate selected API(s)
- Create caching or storage solution
- Build filtering logic (by date, team, league)

## Phase 3: Frontend Setup (3–5 days)
- Create basic layout (calendar/grid)
- Implement league selection (NBA / NFL)
- Implement daily schedule view
- Implement team schedule view

## Phase 4: Testing + Refinement (2–3 days)
- Verify API calls return correct data
- Test filters (team, date, league)
- UX testing — readability, clarity, responsiveness
- Adjust visual layout and color scheme

## Phase 5: Optional Enhancements / Stretch Goals
- Favorite teams highlighting
- Minimal animation or hover effects for game info
- Light/dark theme toggle

## Phase 6: Launch / Submission
- Deploy frontend + backend
- Document APIs used, schema, and decisions
- Prepare demo screenshots or short video
- Submit project / share GitHub repo
